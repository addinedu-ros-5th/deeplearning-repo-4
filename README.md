# deeplearning-repo-4
딥러닝 프로젝트 4조. 객체 탐지, 예측, 회피 시스템

### 목적 및 방향
물류센터에서의 입고부터 출고, 고객 배송까지의 과정을 총망라하는 시스템을 구현한다.\
GUI를 제작하고 GUI로 입력된 내용들은 Data서버에 저장한다.\
I/O를 제작하여 실제 동작을 구현하며, GUI, Server, I/O가 서로 연결되어 원활하고 유기적인 시스템을 만든다.
